# Página de Venta de Ollas Essen y Artículos de Cocina (EN DESARROLLO...)

¡Bienvenido al repositorio de la página web de venta de Ollas Essen y artículos de cocina! Este Readme tiene como objetivo proporcionar una visión general del proyecto y guiar a los desarrolladores.

## Descripción del Proyecto

Este proyecto es una página web dedicada a la venta de ollas Essen y una amplia variedad de artículos de cocina de alta calidad. Nuestra misión es ofrecer a nuestros clientes una experiencia de compra en línea sencilla y atractiva, facilitando la adquisición de productos que mejoren su experiencia culinaria.Dado que por politicas de la empresa no es posible publicar precios, el carrito de compras deriva directamente al Whatsapp del vendedor

## Tecnologías Utilizadas

- **Frontend**:
  - HTML5 y CSS3 para la estructura y el diseño de la página.
  - JavaScript para la interactividad y la experiencia del usuario.
  - Framework de CSS (por ejemplo, Bootstrap) para una apariencia moderna y responsive.

.

## Características Principales

- Catálogo de productos: Mostrar una amplia gama de ollas Essen y artículos de cocina.
- Carrito de compras: Permitir a los usuarios agregar y eliminar productos, así como actualizar cantidades.
- Funcionalidad de búsqueda: Permitir a los usuarios buscar productos específicos.
- Páginas de información: Páginas de contacto.
